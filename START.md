## Как запустить

### 1. Убедитесь что старые процессы остановлены

```bash
pkill -f vite
pkill -f "cargo.*run"
```

### 2. Запустите приложение

```bash
cd /home/vic/noteforge
npm run tauri dev
```

### 3. При первом запуске

Когда откроется окно:
- Вы увидите экран "Welcome to NoteForge"
- Нажмите кнопку "Select Vault Directory"
- Выберите `/home/vic/noteforge-test-vault/`
- После выбора:
  - Слева появится дерево файлов
  - Нажмите на Welcome.md чтобы открыть
  - Попробуйте редактировать текст
  - Автосохранение через 1 секунду

## Если видите синий экран

Откройте консоль разработчика в Tauri (F12 или Ctrl+Shift+I) и проверьте ошибки в Console.

Возможные причины:
1. Vite server не запустился - проверьте терминал
2. JavaScript ошибки - смотрите в консоль браузера
3. Tauri не может подключиться к Vite - проверьте что порт 5173 свободен
