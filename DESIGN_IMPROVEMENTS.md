# Улучшения дизайна NoteForge

## 🎨 Что изменено

### 1. **app.css** — Расширены CSS переменные и глобальные стили

**Добавлено:**
- `--bg-hover` — мягкий hover эффект
- `--bg-elevated` — для приподнятых элементов
- `--accent-softer` — очень мягкий акцент (для активных состояний)
- `--shadow-soft` — мягкая тень
- `--radius-sm/md/lg` — радиусы скругления
- `--transition-fast/normal` — скорости анимаций

**Улучшено:**
- Кнопки — теперь минималистичные, без бордеров, с мягкими hover состояниями
- Добавлен класс `.icon-button` для иконочных кнопок (32px)
- Все transitions — 120-180ms вместо 200ms (более отзывчиво)

---

### 2. **Layout.svelte** — Добавлены toggle кнопки для свернутых панелей

**ИСПРАВЛЕНА ГЛАВНАЯ ПРОБЛЕМА:**
- Когда sidebar или AI panel свернуты — появляются **плавающие кнопки** для их открытия
- Кнопки с иконками-стрелками, позиционированы абсолютно
- Мягкая тень и hover эффект

**Изменения:**
- Sidebar: фиксированная ширина 260px (вместо 250px)
- AI Panel: фиксированная ширина 360px (вместо 350px)
- Добавлены SVG иконки для toggle кнопок

---

### 3. **Sidebar.svelte** — Улучшенный header и иконки

**Изменения:**
- Заголовок "NOTES" — uppercase, letter-spacing, меньший размер
- Иконочные кнопки 28x28px
- SVG иконки вместо текстовых стрелок (+/←)
- Padding в header: 16px
- Border между header и списком: `border-soft` вместо `border`

**Визуально:**
- Более чистый, воздушный header
- Аккуратные иконки

---

### 4. **FileTree.svelte** — Современный дизайн с акцентной полоской

**ГЛАВНОЕ ИЗМЕНЕНИЕ:**
- **Активный файл** — мягкий фон (`--accent-softer`) + тонкая **акцентная полоска слева** (2px)
- Иконки заменены на SVG (стрелки для папок, document icon для файлов)
- Улучшена иерархия — отступ 16px на уровень (вместо 12px)

**Детали:**
- Высота элемента: 32px (вместо произвольной)
- Gap между иконкой и названием: 8px
- Иконки — `--text-muted` по умолчанию, `--accent` для активного
- Hover — мягкий `--bg-hover`
- Folder font-weight: 500 (чуть жирнее файлов)

**Визуально:**
- Чище, современнее, как в Linear/Notion
- Активный файл выделяется мягко, но четко

---

### 5. **EditorPane.svelte** — Улучшенный tab bar и центрирование

**Главное:**
- **Контент редактора центрируется** с max-width: 820px
- Padding: 24px 32px (больше воздуха)

**Tab bar:**
- Иконка document в табе
- Показывается только имя файла (без пути)
- Мягкий фон и border

**Empty state:**
- Красивая иконка SVG
- Два уровня текста: title + hint
- Центрирован вертикально

**Визуально:**
- Как в современных редакторах — текст не растянут на весь экран
- Читаемость улучшена

---

### 6. **AIPanel.svelte** — Табы в стиле Linear

**Табы:**
- Два режима: Chat / Search с иконками
- Active tab — мягкий фон + тень
- Hover эффекты
- Иконки SVG (чат и поиск)

**Header:**
- Аналогично sidebar — uppercase title
- Иконочная кнопка закрытия

**Empty states:**
- Для каждого таба свой placeholder с иконкой
- Круглая обводка + основная иконка
- Title + description

**Визуально:**
- Профессиональный, чистый интерфейс
- Как в Linear или Notion

---

### 7. **Editor.svelte** — Стили для CodeMirror и Markdown

**CodeMirror стилизация:**
- Цвета синхронизированы с темой приложения
- Курсор — `--accent`
- Выделение — `--accent-soft`
- Активная строка — мягкая подсветка

**Markdown типографика:**
- H1: 1.75rem, жирный
- H2: 1.4rem, полужирный
- H3: 1.15rem
- Inline code — фон `--code-bg`, border, акцентный цвет
- Ссылки — цвет accent с мягким underline
- Списки — маркеры в цвете accent

**Gutters:**
- Мягкий border справа
- Номера строк — приглушенные

**Визуально:**
- Markdown хорошо читается
- Иерархия заголовков видна сразу
- Код и ссылки выделяются

## 📊 Общие улучшения

### Spacing & Typography
- Единая система отступов: 4px, 8px, 12px, 16px, 24px, 32px
- Размеры шрифтов: 0.8125rem (13px), 0.875rem (14px), 1rem (16px)
- Line-height: 1.5-1.7 для читаемости

### Цвета
- Убран визуальный шум — меньше ярких акцентов
- Активные состояния — мягкие фоны вместо ярких цветов
- Accent используется дозированно (иконки, полоски, ссылки)

### Интерактивность
- Все transitions: 120-180ms (быстрее, отзывчивее)
- Hover — мягкий фон `--bg-hover`
- Active — `--accent-softer` или `--accent`
- Focus states — border в цвете accent

### Иконки
- Все эмодзи заменены на SVG
- Консистентный стиль (stroke 1.2-1.5px)
- Размеры: 14px, 16px, 20px, 40px, 48px
